# Language Guidelines

## Documentation and Deliverables
All project documentation, code comments, commit messages, and deliverables must be written in **English**.

This includes:
- Code comments
- Documentation files (README, API docs, etc.)
- Commit messages
- System definitions and descriptions
- Test descriptions
- Configuration files
- Spec documents (requirements, design, tasks)

## Communication with AI Assistant
Conversations and interactions with the AI assistant should be conducted in **Japanese** (日本語).

## Code
- Function names, variable names, and all code identifiers: English
- Inline comments: English
- Documentation strings: English
